## 1. 产品概述

山泉水质监测 Web 平台是为景区、水务站和乡镇监管人员打造的水质实时监测与综合管理系统。平台通过接入各监测点的自动传感器数据，结合人工取样录入与巡查管理，实现对山泉水质的全方位监控，保障饮用水源安全。

- 核心目的：实时掌握各水源地水质状况，及时发现并处置异常，留存监管台账
- 目标用户：景区管理人员、水务站值班员、乡镇监管人员
- 市场价值：降低人工巡检成本，提升异常响应速度，满足水务监管合规要求

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 景区管理员 | 管理员分配账号 | 查看所属景区监测数据、接收告警、派发任务 |
| 水务站值班员 | 管理员分配账号 | 查看全部监测数据、录入取样结果、处置告警 |
| 乡镇监管人员 | 管理员分配账号 | 查看辖区数据、巡查记录、导出报表台账 |

### 2.2 功能模块

1. **首页总览**：核心指标卡片、告警摘要、监测点状态分布、最新异常趋势
2. **监测点地图**：地图标注所有监测点位、水源保护范围叠加、点位状态颜色标识
3. **水质详情**：指标时序曲线、按点位筛选、多指标对比、阈值线标注
4. **取样任务**：人工取样表单录入、任务派发与状态跟踪、现场照片上传
5. **告警中心**：异常告警列表、指标阈值配置、告警详情与处置流程填写
6. **巡查记录**：巡查表单填写、现场照片上传、历史巡查记录查询
7. **统计报表**：日报月报生成、指标统计图表、监管台账导出、停报点位提醒

## 3. 核心流程

### 3.1 水质异常处置流程

值班员在首页或告警中心发现异常 → 查看告警详情与指标数据 → 填写处置过程 → 派发复检任务 → 取样人员执行取样 → 录入结果并上传照片 → 确认恢复正常

```mermaid
flowchart TD
    "发现异常告警" --> "查看告警详情"
    "查看告警详情" --> "填写处置过程"
    "填写处置过程" --> "派发复检任务"
    "派发复检任务" --> "执行人工取样"
    "执行人工取样" --> "录入结果+上传照片"
    "录入结果+上传照片" --> "确认恢复正常"
```

### 3.2 日常巡查流程

监管人员发起巡查 → 到达现场检查 → 填写巡查表单 → 上传现场照片 → 提交巡查记录

```mermaid
flowchart TD
    "发起巡查" --> "到达现场"
    "到达现场" --> "填写巡查表单"
    "填写巡查表单" --> "上传现场照片"
    "上传现场照片" --> "提交巡查记录"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝绿色（#0F766E）象征山泉水质，搭配暗色系背景（#0C1222）营造专业监控氛围
- **辅助色**：青色（#06B6D4）用于数据高亮，琥珀色（#F59E0B）用于告警，红色（#EF4444）用于严重异常
- **按钮风格**：圆角微凸按钮，hover态带发光效果
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular，数据使用 JetBrains Mono 等宽字体
- **布局风格**：左侧导航栏 + 顶部标题栏 + 主内容区卡片式布局
- **图标风格**：线性图标（Lucide），搭配微动效

### 4.3 响应式设计

- 桌面优先设计，最小宽度1280px完整体验
- 1024-1280px：侧边栏折叠为图标模式，卡片2列布局
- 768-1024px：顶部导航切换，卡片单列堆叠
- 触屏优化：地图支持双指缩放，表格支持横向滑动
